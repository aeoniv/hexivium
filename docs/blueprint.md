# **App Name**: Hexivium

## Core Features:

- Repository Cloning: Clone the specified GitHub repository to the local file system.
- File System Navigation: Browse the cloned repository's file system and display files and directories in a tree view.
- Code Syntax Highlighting: Render files of certain known filetypes like .js or .py using syntax highlighting to aid readability
- Text search tool: Index all files on the system and implement an AI-powered search for code snippets. The tool decides whether to provide matches based on semantic or syntactic relevance.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke sophistication and technological depth.
- Background color: Dark gray (#333333) for a modern, immersive feel.
- Accent color: Electric blue (#03A9F4) to highlight interactive elements and code snippets.
- Headline font: 'Space Grotesk' sans-serif for headlines, lending a computerized, techy look. Body font: 'Inter' sans-serif for the body.
- Code font: 'Source Code Pro' monospace, optimized for code readability.
- Use minimalist icons to represent file types and actions, following a consistent, monoline style.
- Subtle transitions and animations to indicate loading states and interactive feedback.
